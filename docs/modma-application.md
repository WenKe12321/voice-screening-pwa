## 人工步骤

1. 填写项目名称、负责人、机构、联系方式和研究背景。
2. 勾选建议数据项。
3. 打印并签署 EULA。
4. 扫描为 PDF。
5. 登录 MODMA 官网并上传。
6. 等待管理员审批。

## 许可边界

- 仅可用于学术研究，不得用于商业用途。
- 不得分发数据集或其片段。
- 不得共享账号和密码。
- 论文和展示需要按 EULA 引用兰州大学来源与指定论文。
- 在未取得书面许可前，不公开上传基于 MODMA 训练的派生权重。
