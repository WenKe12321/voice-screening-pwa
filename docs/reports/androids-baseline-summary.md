# Androids 独立五折基线汇总

## 结论

已使用真实 Androids Corpus 完成朗读任务和访谈任务的独立五折实验。Androids 使用患者/健康对照条件标签，不是 SDS、PHQ-9 或临床诊断结论。实验结果不能与 EATD 混算，也不会导出为手机端模型。

## 数据与协议

- 数据集：Androids Corpus。
- 归档 SHA-256：`f8490307fe24029ff5c14147dcced360f65b0c211524f54b59f850c26fb70502`。
- 朗读任务：`112` 名受试者。
- 访谈任务：`116` 名受试者。
- 划分：压缩包内作者提供的 `fold-lists.csv`。
- 方法：按任务分别执行五折受试者隔离评估，训练折内部交叉验证调参和选阈值。
- 特征：与浏览器端一致的语音比例、停顿、RMS、过零率、基频、频谱质心、MFCC 和语速代理指标。

## 默认阈值对照

使用默认阈值 `0.5` 时，访谈逻辑回归的 ROC-AUC 为 `0.8996`，召回率为 `0.8906`，特异度为 `0.8269`。这些指标仅用于离线研究比较，不能替代面向目标人群的外部验证。

## 边界

1. Androids 不是中文大学生样本，不能证明“心声自测”对目标用户有效。
2. Androids 标签不能解释为 PHQ-9 风险等级。
3. 朗读和访谈结果不得与 EATD 混合计算准确率。
4. 原始语音、派生特征和逐样本预测只保存在 `F:\Datasets\voice-screening`。
