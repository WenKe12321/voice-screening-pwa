# EATD 中文开放回答基线汇总

## 结论

已使用真实 EATD-Corpus 数据完成端侧声学特征基线训练。当前逻辑回归模型没有达到手机应用启用门槛，因此没有生成可导入的真实 `.vmodel`。

这意味着当前版本只能继续使用“未验证演示指数”。不能将候选模型包装为可用筛查模型，也不能宣称具备临床准确率。

## 数据与划分

- 数据集：EATD-Corpus。
- 受试者：`162` 人。
- 训练集：作者提供的 `83` 人。
- 验证集：作者提供的 `79` 人。
- 标签：标准 SDS 分数 `>= 53`。
- 特征：与浏览器端一致的停顿、RMS、过零率、基频、频谱质心、MFCC 和语速代理摘要。
- 异常处理：`1` 段 `_out.wav` 为空，按审计规则回退到对应原始 `.wav`。
