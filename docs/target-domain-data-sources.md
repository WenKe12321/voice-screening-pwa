# 中文大学生目标域语音数据来源

## 结论

当前尚未取得可直接用于本项目训练的、带明确授权范围的中文大学生 `PHQ-9` 目标域录音。

仓库不会收录原始语音、参与者级元数据、派生权重或逐样本预测。获批数据统一保存在：

```text
F:\Datasets\voice-screening
```

在申请或采集任何非公开录音前，需要由项目负责人完成机构伦理审查，确认知情同意范围，并保存书面许可。中国政府网发布的《涉及人的生命科学和医学研究伦理审查办法》明确将高校使用人的健康、行为等信息数据开展研究纳入伦理审查范围。

## 来源分级

### 第一优先级：天桥脑科学研究院大学生筛查数据集

官方数据服务页列出“大学生抑郁焦虑筛查数据集”，包含 `PHQ-9`、`BDI-II`、`GAD-7`、`SAS` 四种量表，以及朗读、自发言语等语音数据。它与本项目目标人群最接近。

- 数据页：[天桥脑科学研究院数据集](https://www.dlab.org.cn/data-set)
- 联系页：[联系我们](https://www.dlab.org.cn/contact-us)
- 联系邮箱：`sdf@cheninstitute.org`
- 状态：需要人工提交咨询或发送申请邮件。

建议申请时确认：

1. 是否可提供去标识化语音、`PHQ-9` 总分和题目级得分。
2. 是否可提供录音任务文本、采集设备和环境说明。
3. 是否允许用于非商业学术研究、模型训练和去标识化汇总报告。
4. 是否允许仅在本地保存派生权重，以及是否允许发布权重。
5. 是否存在校区、年龄、性别等可用于分层验证的字段。

### 第二优先级：福建中医药大学阈下抑郁朗读语音

陈梅妹等人的论文采集了大学生朗读十个中性词和《北风与太阳》的语音，与当前普通自测流程对齐。论文使用 `CES-D` 和 `HAM-D` 纳入标准，而不是 `PHQ-9`，因此更适合作为任务对齐评估集。

- 论文：[基于多种机器学习算法和语音情绪特征的阈下抑郁辨识模型构建](https://cjournal.hep.com.cn/1673-4254/CN/1160172233005130687)
- 联系邮箱：`chenmeimei1984@163.com`、`yzy813@126.com`
- 状态：论文未公开原始录音下载链接，需要取得作者书面许可。

### 第三优先级：东南大学 SubCD

东南大学实验室公开介绍了 `SubCD` 多模态行为数据集，包含访谈、图片描述和视频描述任务下的语音、文本和表情数据。它可能适合作为单独报告的外部验证集。

- 介绍页：[东南大学实验室项目说明](https://aip.seu.edu.cn/2025/0622/c54088a534051/page.htm)
- 状态：公开页面没有下载入口。需要联系研究团队并确认参与者范围、标签定义、伦理范围和许可条款。

### 已有基线：EATD-Corpus

本机已保存 EATD-Corpus。作者仓库说明它包含 162 名志愿者的中文音频和文本，并给出公开下载方式。它可继续作为中文学生志愿者基线，但使用 SDS 自评标签和三个情绪开放回答任务。

- 作者仓库：[ICASSP2022-Depression](https://github.com/speechandlanguageprocessing/ICASSP2022-Depression)
- 状态：本机已下载和校验。
- 限制：不能替代中文大学生 `PHQ-9` 目标域验证。

### 临床外部验证：CMDC 与 MODMA

CMDC 和 MODMA 均可用于单独报告的中文临床外部验证，但不能替代大学生目标域训练。

- CMDC：[CMDC-Baseline](https://github.com/CMDC-corpus/CMDC-Baseline)，需要签署 EULA 并向维护者申请口令。
- MODMA：[官方申请页](https://modma.lzu.edu.cn/data/application/)，需要注册机构邮箱、签署并上传 EULA，等待管理员授权。
